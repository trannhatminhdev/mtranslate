# 📘 Mtranslate

## ▶️ Hướng dẫn

### Bước 1: Tải VB-CABLE Virtual

VB-CABLE Virtual dùng để tạo micro ảo.  
Link: [https://vb-audio.com/Cable/](https://vb-audio.com/Cable/)

---

### Bước 2: Chọn thiết bị  

Chạy mtranslate.exe, chọn thiết bị là: CABLE Input (VB-audio Virtual Cable)
